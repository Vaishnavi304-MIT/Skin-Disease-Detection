# 🧠 Skin Disease Detection Using Vision Transformer (ViT)

> **An AI-powered system that detects and classifies skin diseases using deep learning and transformer-based architecture for accurate and early diagnosis.**

## 🚩 Problem Statement
Skin diseases are common and often difficult to diagnose accurately without expert consultation.  

Traditional diagnostic methods such as biopsy, dermoscopy, and patch testing:
- Are time-consuming  
- Require specialized medical professionals  
- Can delay treatment  

---

## 💡 Solution
An AI-based system that uses a **Vision Transformer (ViT)** model to analyze skin images and classify diseases automatically.

- Provides **fast and accurate predictions**  
- Reduces dependency on manual diagnosis  
- Helps in **early detection and treatment**  

---

## 🛠️ Technologies Used
- Deep Learning  
- Computer Vision  
- Vision Transformer (ViT)  
- Python  
- NumPy, Pandas  
- Matplotlib / OpenCV  

---

## ⚙️ Model Details
- **Model:** Vision Transformer (ViT)  
- **Input Size:** 224 × 224 images  
- **Dataset:** DermNet (~16,000 images)  
- **Classes:** 23 skin diseases  
- **Type:** Multi-class classification  

---

## ✨ Wow Factors
- Uses **Transformer instead of traditional CNN**  
- Captures **global image features using self-attention**  
- Handles **complex and similar disease patterns**  
- Provides **high accuracy (~70%) on multi-class dataset**  
- Reduces confusion between visually similar diseases

## 🧠 Methodology

### 🔹 Data Preprocessing
- Image resizing (224×224)  
- Normalization  
- Data augmentation (flip, rotation, zoom)  

---

### 🔹 Vision Transformer Workflow
1. Image split into patches  
2. Patch embedding  
3. Positional encoding added  
4. Transformer encoder (self-attention)  
5. Classification using Softmax  

---

## 📊 Results
The model successfully classifies skin diseases with good accuracy and handles complex datasets efficiently.
